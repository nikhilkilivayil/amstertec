BIN TOP

Bin Tops are GPS enabled devices which can be attached to any bin including large waste containers that are capable of detecting waste fill level and send real time data to Cloud Collection Connect. Fill level sensors helps in reducing number of collections and collection cost.

How Bin Top works?

  • Bin Top can be attached to any type of bin irrespective of size. The ultrasonic fill level sensors within the device detect the waste level.
  • The data send will be monitored by cloud based system.
  • Once the waste reaches peak level, data send by the sensor will notify the collection agents. This will prevent the waste spillage from the bins and getting accumulated around the bin.
  • Bin Tops are equipped with anti theft and GPS tracking facilities, which helps to track the devices.
  • No need of replacing your current bin, instead install Bin Tops and prevent waste overflow and litter accumulation.
